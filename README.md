# 🛍️ ShopWise AI – Your Intelligent Shopping Companion

## 📌 Project Overview

**ShopWise AI** is an AI-powered shopping assistant designed to simplify online shopping by helping users make informed purchasing decisions. The system compares product prices, analyzes customer reviews, detects fake reviews, predicts future price trends, and provides personalized product recommendations using **IBM Granite Models**, **IBM watsonx.ai**, **LangFlow**, and **Retrieval-Augmented Generation (RAG)**.

The project aims to reduce the time users spend comparing products across multiple e-commerce platforms while ensuring they purchase the best product at the best possible price.

---

# 🎯 Problem Statement

Online shoppers often struggle with:

- Comparing prices across multiple shopping websites.
- Identifying genuine customer reviews.
- Choosing products based on their budget and preferences.
- Predicting future price drops.
- Finding the best value-for-money products.

ShopWise AI solves these problems by acting as an intelligent shopping companion.

# 🧠 Agentic AI Workflow

1. User enters a shopping query.
2. LangFlow orchestrates multiple AI agents.
3. IBM Granite processes the request.
4. Product data is retrieved using RAG.
5. AI compares prices and analyzes reviews.
6. Personalized recommendations are generated.
7. Dashboard displays shopping insights.

---

# 👨‍💻 AI Agents

### 🛒 Product Recommendation Agent
Recommends products based on user preferences, budget, and ratings.

### 💰 Price Comparison Agent
Compares prices from multiple e-commerce platforms.

### ⭐ Fake Review Detection Agent
Analyzes reviews to detect spam or fake feedback.

### 📈 Price Prediction Agent
Predicts future price trends to recommend the best buying time.

### 👤 User Preference Agent
Learns user preferences to deliver personalized recommendations.
